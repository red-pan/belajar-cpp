## Many ways to print output to the console
*    ### `std::cout`
     `cout` (console out)
*    ### `std::clog`
     `clog` (console log)
*    ### `std::cerr`
     `cerr` (console error)
*    ### `printf()`
     `printf()` function is used to write a string to the standard output (`stdout`).
     #### `printf()` Syntax:
         printf(const char* format, ...)
